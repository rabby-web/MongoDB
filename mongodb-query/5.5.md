## Logical Query Operators

- **`$and`** (Logical AND):

  ```javascript
  db.collection.find({
    $and: [{ age: { $gte: 18 } }, { gender: "Female" }],
  });
  ```

- **`$not`** (Negation):

  ```javascript
  db.collection.find({ age: { $not: { $gte: 18 } } });
  ```
