## `$set more documentation explore`

**`$set`**

```javascript
// update a single object field with $set
db.test.updateOne(
  { _id: ObjectId("6406ad63fc13ae5a40000065") },
  {
    $set: {
      "address.city": "Dhaka",
    },
  }
);
// update a multiple object field with $set
db.test.updateOne(
  { _id: ObjectId("6406ad63fc13ae5a40000065") },
  {
    $set: {
      "address.city": "Dhaka",
      "address.country": "Bangladesh",
    },
  }
);
```
