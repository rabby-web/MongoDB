## `$match,$project aggregate pipeline`

```js
// mongoDB query
db.test
  .find({
    gender: "Male",
  })
  .project({ name: 1, gender: 1, age: 1 });
  
//  mongoDB aggregation 
db.test.aggregate([
    // stage:-1
    {
        $match: {
            gender: "Male",
            age: { $lt: 30 }
        }
    },
    // stage:-2
    {
        $project: {
            name: 1,
            age: 1,
            gender: 1
        }
    }
])
```
